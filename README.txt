Udacity Carnival Virtual Reality Application
VR Equipment: Google Cardboard

One primary scene where the player can interact and play 3 games for a classic carnival prize.
Games:
> Plinko
> Coin Toss
> Wheel of Fortune

Prize:
> Teddy Bear (once the player accumulates 2000 pts.)

Software Used:
GVR Unity SDK v1.60.0
Unity 2017.1.0p4 (64 bit)
Android SDK API 24

Deployed to:
Samsung Galaxy S6
